File Handling in Python

File handling in Python allows you to work with files, both for reading and writing data. Here's how you can use file handling operations:

  1. Opening a File

    2. To work with a file, you need to open it using the open() function. You specify the file name and the mode in which you want to open the file (read, write, or both).

    # Opening a file for reading
file = open("example.txt", "r")

# Opening a file for writing
file = open("output.txt", "w")
  2. Reading from a File

    3. You can read the contents of a file using methods like read(), readline(), or readlines(). 

    # Reading the entire content
content = file.read()

# Reading one line
line = file.readline()

# Reading all lines into a list
lines = file.readlines()
  3. Writing to a File

    4. You can write to a file using methods like write().

    # Writing a string to the file
file.write("Hello, world!")
  4. Closing a File

    5. It's important to close the file after you're done working with it to free up system resources.

    file.close()
  5. Using 'with' Statement (Context Manager)

    6. The with statement ensures that the file is properly closed after you're done, even if an exception occurs.

    with open("example.txt", "r") as file:
content = file.read()
  6. Appending to a File

    7. To add content to an existing file without overwriting it, you can open the file in append mode ("a"). 

    with open("output.txt", "a") as file:
file.write("Additional content")

    File handling is a fundamental aspect of programming for tasks involving data storage, retrieval, and manipulation. Always make sure to properly close the file after you're done with it to avoid resource leaks.
