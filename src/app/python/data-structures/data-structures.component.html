Built-In Data Structures in Python

Python offers several built-in data structures that help you organize and manipulate data efficiently. Here are some of the main data structures:

  1. Lists

    2. Ordered, mutable collections of items.

    my_list = [1, 2, 3, 4]
my_list.append(5)
my_list.pop(2)
  2. Tuples

    3. Ordered, immutable collections of items.

    my_tuple = (1, 2, 3)
value = my_tuple[1]
  3. Sets

    4. Unordered collections of unique items.

  4. Dictionaries

    5. Unordered collections of key-value pairs.

  5. Strings

    6. Sequences of characters.

    my_string = "Hello, Python"
substring = my_string[7:13]
  6. Lists of Lists (Nested Lists)

    7. Creating lists that contain other lists.

    matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
value = matrix[1][2]
  7. List Comprehensions

    8. Creating lists using a compact syntax.

    squares = [x ** 2 for x in range(1, 6)]
