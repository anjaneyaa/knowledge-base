String Manipulation in Python

String manipulation in Python involves various operations to work with and manipulate strings. Here are some common string manipulation techniques:

  1. Concatenation

    2. Combining strings together.

    str1 = "Hello"
str2 = "World"
result = str1 + " " + str2
  2. String Formatting

    3. Creating formatted strings using placeholders.

  3. String Methods

    4. Python provides a wide range of string methods for various operations.

    my_string = "Hello, Python!"
length = len(my_string)
uppercase = my_string.upper()
lowercase = my_string.lower()
  4. String Slicing

    5. Extracting portions of a string.

    my_string = "Hello, Python!"
substring = my_string[7:13]  // Extracts "Python"
  5. String Splitting and Joining
    6. my_string = "apple,banana,orange"
fruits_list = my_string.split(",")

fruits = ["apple", "banana", "orange"]
joined_string = "-".join(fruits)
  6. Stripping Whitespaces
    7. my_string = "   Hello, Python!   "
stripped_string = my_string.strip()  // Removes leading and trailing whitespaces
  7. Replacing Substrings
    8. my_string = "Hello, World!"
new_string = my_string.replace("World", "Python")
  8. Checking Substrings
    9. my_string = "Hello, Python!"
contains_python = "Python" in my_string
  9. String Formatting (f-strings) (Python 3.6+)

    10. String manipulation is essential for working with textual data and is used extensively in applications ranging from data processing to web development.
