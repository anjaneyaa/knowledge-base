Basic Syntax and Data Types in Python
  1. Variables: Used to store data. No need to declare a variable's type explicitly.
  2. Data Types: Python has various built-in data types:
    • Integers: Whole numbers.
    • Floats: Numbers with decimal points.
    • Strings: Sequences of characters.
    • Boolean: Represents True or False.
  3. Operators: Used to perform operations on variables and values.
    • Arithmetic Operators: +, -, *, /, %, // (floor division), ** (exponentiation).
    • Comparison Operators: ==, !=, <, >, <=, >=.
    • Logical Operators: and, or, not.
  4. Basic Input/Output: Using input() to get user input and print() to display output.
  5. Comments: Lines beginning with # are comments and are ignored by the interpreter.

    6. Here's a simple example of Python code that demonstrates these concepts:

    python 
    
        # This is a comment
        num1 = 10           # An integer variable
        num2 = 3.14         # A float variable
        name = "John"       # A string variable
        is_student = True   # A boolean variable
        # Arithmetic operations
        sum_result = num1 + num2
        product_result = num1 * num2
        # Displaying output
        print("Hello,", name)
        print("Sum:", sum_result)
        print("Product:", product_result)
        # Getting user input
        age = input("Enter your age: ")
        print("You are", age, "years old.")
Control Structures in Python

Control structures in Python allow you to control the flow of your program. Here are the main control structures:

  1. Conditional Statements (if, elif, else)
    2. Used to execute different code blocks based on conditions.
    python 
    if condition1:
            # Code to execute if condition1 is True
            elif condition2:
            # Code to execute if condition2 is True
            else:
            # Code to execute if none of the conditions are True
  2. Loops (for, while)
    3. Used to execute a block of code repeatedly.
    • For Loop

      • Iterates over a sequence (list, tuple, string, etc.).

      for item in sequence:
    # Code to execute for each item in the sequence
    • While Loop

    Executes a block of code as long as a condition is True.

    while condition:
    # Code to execute while the condition is True
  3. Break and Continue Statements

    4. Used to modify the flow of loops.

    • Break

      • Exits the loop prematurely.

      for item in sequence:
    if condition:
        break
    • Continue

      • Skips the current iteration and moves to the next.

      for item in sequence:
    if condition:
        continue

      Here's an example demonstrating these control structures:

      python 
         # Conditional Statements
            x = 10
            if x > 0:
                print("x is positive")
            elif x == 0:
                print("x is zero")
            else:
                print("x is negative")

    # For Loop
            numbers = [1, 2, 3, 4, 5]
            sum = 0
            for num in numbers:
                sum += num
            print("Sum:", sum)

            # While Loop
            count = 0
            while count < 5:
                print("Count:", count)
                count += 1

            # Break and Continue
            for i in range(10):
                if i == 3:
                    break
                if i % 2 == 0:
                    continue
                print(i)

    Control structures help you create more dynamic and responsive programs by allowing you to make decisions and repeat actions as needed.

Functions in Python

Functions in Python are reusable blocks of code that perform specific tasks. They help organize code, make it more modular, and reduce redundancy. Here's how to work with functions:

  1. Defining a Function

    2. To define a function, use the def keyword followed by the function name and parameters within parentheses. The function body is indented below.

    def greet(name):
            print("Hello, " + name)
  2. Calling a Function

    3. To use a function, call it by its name and provide arguments if needed.

    greet("Alice")
  3. Parameters and Arguments

    4. Parameters are variables in a function's definition. Arguments are values passed to a function when it's called.

    def add(a, b):
             return a + b
    
            result = add(3, 5)  # Here, 3 and 5 are arguments
  4. Return Statement

    5. Functions can return values using the return statement. If no return value is specified, the function returns None.

    def square(x):
            return x * x
    
            squared_value = square(4)
  5. Default Arguments

    6. You can provide default values for function parameters, which are used when the caller doesn't provide a value for that parameter.

    def power(base, exponent=2):
            return base ** exponent
            
            result1 = power(3)      # Uses default exponent of 2
            result2 = power(2, 3)   # Uses provided exponent of 3
  6. Variable Number of Arguments

    7. Functions can accept a variable number of arguments using *args for positional arguments and **kwargs for keyword arguments. 

    def print_args(*args):
    for arg in args:
        print(arg)
    
print_args(1, 2, 3)
  7. Scope

    8. Variables defined inside a function are in its local scope. Variables defined outside functions are in the global scope.

    global_var = 10  // Global variable
    
def demo_scope():
    local_var = 5  // Local variable
    print(global_var)
    print(local_var)
    
demo_scope()

    Functions are a fundamental building block of Python programming, allowing you to create more organized, reusable, and maintainable code.
