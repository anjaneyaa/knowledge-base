Control Structures in Python

Control structures in Python allow you to control the flow of your program. Here are the main control structures:

  1. Conditional Statements (if, elif, else)
    2. Used to execute different code blocks based on conditions.
  2. Loops (for, while)
    3. Used to execute a block of code repeatedly.
    • For Loop

      • Iterates over a sequence (list, tuple, string, etc.).

    • While Loop

      • Executes a block of code as long as a condition is True.

  3. Break and Continue Statements

    4. Used to modify the flow of loops.

    • Break

      • Exits the loop prematurely.

    • Continue

      • Skips the current iteration and moves to the next.
